# Deep Space Studio Control Center

A premium, high-fidelity production management system for modern music studios and record labels. Built with **Angular 21**, this platform utilizes a sleek glassmorphism aesthetic to provide a streamlined workflow for project tracking, asset management, and legal clearance.

## 🚀 Key Features

- **Studio Control Center**: Real-time project tracking from Pre-Production to Release.
- **Asset Library**: High-density management of audio samples and recording assets.
- **Legal Clearance Vault**: Automated rights management and authorization ledger for sample usage.
- **Premium UI/UX**: Custom glassmorphism design with responsive grid layouts and subtle animations.

## 🛠 Tech Stack

- **Framework**: Angular 21 (Standalone Components, Signals architecture)
- **Styling**: Bootstrap 5 + Custom Glassmorphism CSS Tokens
- **Icons**: Bootstrap Icons
- **Testing**: Vitest

## 💻 Getting Started

### Development Server

To start the local production environment, run:

```bash
pnpm start
```

Once the server is running, open your browser and navigate to `http://localhost:4200/`. The application will automatically reload whenever you modify any of the source files.

### Code Scaffolding

To generate a new studio logic or UI component, run:

```bash
ng generate component component-name
```

## 🏗 Building for Production

To compile the project for distribution:

```bash
ng build
```

The build artifacts will be stored in the `dist/` directory, optimized for performance and security.

## 🧪 Quality Assurance

### Running Unit Tests

To execute the test suite with the [Vitest](https://vitest.dev/) runner:

```bash
ng test
```
